# CAPTCHAS Skills and OpenClaw Integrations

Ready-to-use skills and examples for adding CAPTCHAS human verification to agent workflows.

## What’s in this repo
- `captchas-human-verification/` — core skill for Agent API workflows
- `captchas-openclaw/` — OpenClaw-specific tool schemas and plugin guidance
- `captchas-human-verification/examples/` — integration folders (Zendesk, Telegram, Slack)

## Quick start
1. Pick a skill:
   - `captchas-human-verification/SKILL.md`
   - `captchas-openclaw/SKILL.md`
2. Set your environment:
   - `CAPTCHAS_ENDPOINT=https://agent.captchas.co`
   - `CAPTCHAS_API_KEY=<your-api-key>`
3. Follow the workflow steps in the skill.
